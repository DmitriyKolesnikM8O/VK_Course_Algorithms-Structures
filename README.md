# Автоматизированный сканер портов и уязвимостей

## Описание проекта
Программное решение для мониторинга защищаемого периметра. Скрипт в автоматическом режиме находит открытые порты, определяет запущенные сервисы, анализирует их на наличие известных уязвимостей (CVE) и уведомляет владельца о новых угрозах в Telegram.

## Основной функционал
- **Быстрое сканирование:** Использование `Masscan` (до 1000 пакетов в секунду).
- **Глубокий анализ:** Интеграция с `Nmap` (NSE скрипты) для определения версий ПО и поиска CVE.
- **Интеллектуальные уведомления:** Система сообщает только о новых портах, которых не было в базе данных.
- **Персистенция:** История сканирований хранится в асинхронной БД `SQLite`.
- **Асинхронность:** Полностью неблокирующая архитектура на `Python asyncio`.

## Алгоритм работы
1. **Разведка (Recon):** Masscan проверяет заданные IP/диапазоны/AS.
2. **Сравнение:** Полученные результаты сверяются с БД SQLite.
3. **Анализ (Deep Scan):** Для новых портов запускается Banner Grabbing и Nmap-анализ.
4. **Оповещение:** Сформированный отчет отправляется через Telegram API.
5. **Запись:** Новые угрозы сохраняются в историю.

## Требования
- Python 3.10+
- Masscan
- Nmap
- Установленные зависимости из `requirements.txt`

## Настройка
Все параметры (цели, порты, API токены) настраиваются в файле `config/config.yaml`.
